 # 🌐 Blog Website

A modern and responsive **Blog Website UI** built using **HTML & CSS**. This project features a clean card-based layout to display blog posts with images, author details, and categories.

---

## ✨ Features

* 📱 Responsive design (mobile, tablet, desktop)
* 🧩 Card-based blog layout
* 🖼️ Image integration
* 👤 Author details with date
* 🎨 Smooth hover effects
* 🏷️ Blog category tags

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)

---

## 📁 Project Structure

```
Blog-Website/
│── Index.html
│── Style.css
│── Media/
│    ├── images
│    ├── logo
```
